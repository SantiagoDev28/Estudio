# Tipos de datos en Java.

## Tipos de datos primitivos

Palabras reservadas para los tipos de datos primitivos

- Enteros: `byte`, `short`, `int` y `long`
- Decimales: `float` y `double`
- Caracteres: `char`
- Booleanos: `boolean`.

## Tipos por referencia (Objetos)

Datos que provee java

- Referencias
- Clases
- String
- Objetos
- Arrays

Manejo de estos datos de refencia se encuentra en la seccion de POO (Programacion Orientada a Objetos).

# Casting

Casting es la accion de convertir un tipo de datos en otro.

## Casting implicito

Casting implicito es la conversion de un tipo de dato mas grande a un tipo de dato mas pequeno.

## Casting explicito

Casting explicito es la conversion de un tipo de dato mas pequeno a un tipo de dato mas grande.
