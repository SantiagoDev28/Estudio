# Spring-Boot.

Este modulo documenta mi aprendizaje basado en el framework de Java Spring-boot
Aqui se encontraran mis practicas creando una API completa aplicando las mejores practicas de desarrollo de software.

## Objetivo General

Adquirir dominio práctico de Spring Boot para desarrollar aplicaciones backend profesionales, orientadas a entornos productivos, empleabilidad y creación de proyectos propios escalables.

## Objetivos Específicos

- Comprender la arquitectura y filosofía de Spring Boot y su relación con el Spring Framework.

- Configurar y estructurar proyectos Spring Boot siguiendo estándares profesionales de la industria.

- Desarrollar APIs REST robustas utilizando Spring MVC.

- Aplicar correctamente Inyección de Dependencias y principios de bajo acoplamiento.

- Persistir datos utilizando Spring Data JPA y ORM (Hibernate).

- Diseñar modelos de dominio claros usando entidades, DTOs y repositorios.

- Gestionar configuraciones mediante application.properties / application.yml y perfiles.

- Implementar manejo centralizado de errores y respuestas HTTP consistentes.

- Comprender el ciclo de vida de la aplicación Spring Boot y su arranque.

- Introducir buenas prácticas de testing básico con Spring Boot.

- Preparar aplicaciones para entornos reales (puertos, perfiles, empaquetado y ejecución).
